# PowerBI-DataModel-Project
##Project Overview
This project demonstrates dimensional modeling in Power BI using customer and product datasets. The objective is to create relationships between tables and build a structured data model for analysis.
**Dataset Used**
Dim Customers
Dim Products
Dim Promotions
sheet 3 used as fact tables

**Steps Performed**
**Step 1 : Import Data**
Imported Excel datasets into Power BI
Loaded all pages which are in Excel sheet

**Step 2: Data Cleaning**
Click on Transform data
Checked for null values
Removed Duplicate records
Verified and corrected data types
Add new columns if required

**step 3: Data Modeling**
Created relationships between dimension tables and fact tables
Applied one-to-many relationships
Implemented a Star Schema structure

**step 4: DAX Calculations**
Created measures using DAX functions
Used aggregate functions for analysis

**Step 5: Dashboard Creation**
Added charts and tables
Applied slicers and filters
Designed an interactive report page

**Skills Demonstrated**
Power BI
Power Query
Data Modeling
DAX
Data visualization
